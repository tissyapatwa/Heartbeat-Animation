# ❤️ Animated Heart Canvas

A beautiful and interactive heart animation created using HTML5 Canvas, CSS, and JavaScript. The project features a glowing heart with smooth heartbeat effects, animated particles, and a responsive full-screen design.

## ✨ Features

- ❤️ Animated heart shape
- 💓 Smooth heartbeat effect
- ✨ Particle animation
- 🌌 Beautiful dark background
- 📱 Responsive full-screen layout
- 🎨 Built completely with HTML5 Canvas

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas API

## 📂 Project Structure

```
Animated-Heart-Canvas/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Animated-Heart-Canvas.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
